## Setup dev env

### Prerequisites

1. NodeJS & yarn (NPM should also work but we are maintaining a yarn lock file only
2. docker and docker-compose (for postgres)

### Steps
* install dependencies

`yarn install`
* to run the application

`yarn start`
* to build and package the application

`yarn package`
