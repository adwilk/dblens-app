## Setup dev env

### Prerequisites

NodeJS & yarn (NPM should also work but we are maintaining a yarn lock file only

### Steps
* install dependencies

```
yarn install
```
* to run the application

```
yarn start
```
* to build and package the application

```
yarn package
```
